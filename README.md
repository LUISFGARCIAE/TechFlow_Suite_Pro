🚀 TechFlow Suite Pro v4.0
The Ultimate IT Support & Automation Toolkit
TechFlow Suite Pro es una solución integral de automatización diseñada para especialistas en soporte TI. Optimiza el flujo de trabajo desde el despliegue inicial (Post-Format) hasta el mantenimiento avanzado y rescate de sistemas Windows.

✨ Características de la Versión 4.0
📦 Motor de Instalación Inteligente (Híbrido)
Kit Post-Format (100+ Apps): Instalación masiva mediante un motor que alterna entre Winget y Chocolatey. Si uno falla, el otro toma el relevo.

Perfiles de Usuario: Instalación un clic para perfiles Básico, Gaming o Selección Manual mediante IDs.

Gestor de Bloatware: Limpieza profunda de Apps preinstaladas (Candy Crush, TikTok, Disney+, etc.).

🛠️ Herramientas de Soporte Pro
Bypass Windows 11: Salta las restricciones de TPM 2.0, Secure Boot y RAM. Incluye bypass de cuenta Microsoft/Internet (OOBE).

Gestión de Drivers: Exportación e importación masiva de controladores .inf (Ideal para formateos rápidos).

Rescate de Sistema: Reparación de imagen (DISM/SFC), sincronización de hora NTP y recuperación de Clave de Producto BIOS (OEM).

⚡ Optimización y Monitoreo Real
Live Monitor PRO: Panel en tiempo real para CPU/RAM y gestión de procesos (Kill Task) directamente desde la consola.

Mantenimiento de Discos: Comandos específicos para SSD (ReTrim) y HDD (Defrag).

Control Total de Defender: Script de desactivación/activación rápida para entornos de mantenimiento.

🛡️ Seguridad y Backup
Motor Robocopy: Backups de perfil de usuario ultra-rápidos con soporte multihilo (/MT:16).

PIN de Seguridad: Confirmación obligatoria mediante código aleatorio para operaciones críticas (Formateo de unidades y purga de archivos).

Suite de Red: Reseteo de DNS/IP, monitor de Ping, test de velocidad y recuperación de claves Wi-Fi.

🚀 Instalación y Uso
Descarga: Clona el repositorio o descarga el .exe compilado.

Ejecución: Ejecuta siempre como Administrador para garantizar el acceso al registro y servicios.

Interfaz: Usa la tecla V para alternar entre el menú Horizontal (compacto) o Vertical (detallado).
